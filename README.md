
# BitMap

O BitMap é uma estrutura de dados desenvolvida em C que realiza operações semelhantes a um Map usando baixo consumo de memória ideal para aplicações em sistemas embarcados.   


# Interface 

- size()
- set() 
- unset()
- get()
